# Visual Studio Code Keyboard Shortcuts

## Default shortcuts for Windows and macOS

| Action | Windows | macOS |
|---|---|---|
| New Window | `Ctrl+Shift+N` | `Cmd+Shift+N` |
| Close Window | `Ctrl+Shift+W` | `Cmd+Shift+W` |
| Close Editor | `Ctrl+W` | `Cmd+W` |
| Open File | `Ctrl+O` | `Cmd+O` |
| Save File | `Ctrl+S` | `Cmd+S` |
| Save All | `Ctrl+K S` | `Cmd+Option+S` |
| Open Command Palette | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| Toggle Sidebar | `Ctrl+B` | `Cmd+B` |
| Toggle Panel | `Ctrl+J` | `Cmd+J` |
| Toggle Integrated Terminal | `` Ctrl+` `` | `` Ctrl+` `` |
| New Terminal | `Ctrl+Shift+` `` | `Cmd+Shift+` `` |
| Split Editor | `Ctrl+\` | `Cmd+\` |
| Focus 1st Editor Group | `Ctrl+1` | `Cmd+1` |
| Focus 2nd Editor Group | `Ctrl+2` | `Cmd+2` |
| Focus 3rd Editor Group | `Ctrl+3` | `Cmd+3` |
| Toggle Full Screen | `F11` | `Ctrl+Cmd+F` |
| Toggle Zen Mode | `Ctrl+K Z` | `Cmd+K Z` |
| Open Recent | `Ctrl+R` | `Cmd+R` |
| Find | `Ctrl+F` | `Cmd+F` |
| Replace | `Ctrl+H` | `Cmd+Option+F` |
| Find in Files | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| Replace in Files | `Ctrl+Shift+H` | `Cmd+Shift+H` |
| Go to File | `Ctrl+P` | `Cmd+P` |
| Go to Line | `Ctrl+G` | `Cmd+G` |
| Toggle Comment | `Ctrl+/` | `Cmd+/` |
| Format Document | `Shift+Alt+F` | `Shift+Option+F` |
| Rename Symbol | `F2` | `F2` |
| Show Problems | `Ctrl+Shift+M` | `Cmd+Shift+M` |
| Open Settings | `Ctrl+,` | `Cmd+,` |
| Open Extensions | `Ctrl+Shift+X` | `Cmd+Shift+X` |

> Note: These are the default Visual Studio Code keybindings. Custom keybindings or installed extensions may change the actual shortcuts.
